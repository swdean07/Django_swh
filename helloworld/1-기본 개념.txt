Django : 쟁고, 장고라 부름.

파이썬으로 작성된 웹 프레임워크

기능
1. 데이터베이스 관리, 저장, 불러오기, 일관적이고 쉬운편.

2. 관리자 기능.
기본적으로 제공. 개발자가 따로 만들지 않아도, 사용하기 편함.
crud 가능하고, 조금 수정하면, 실제 운영에도 사용이 가능함.

3. 보안.
장고에서 제공하는 기본 보안 기능정도를 사용해도 괜찮은 수준.
따로 off 하지 않는이상, 사용할 만하다.
개발자가 따로 신경 안써도 될만큼.

4. 다양한 내장 기능과 파이썬 확장 기능.
딥러닝 프레임워크로 모델 생성. -> 모델 파일 형식 내보내기
이 모델을 사용하려면,
-> ai 모델을 이용하기 위해서, 장고를 사용함.
